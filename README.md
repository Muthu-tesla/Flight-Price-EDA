# Flight Price EDA Project

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Flight Price dataset to understand how different features affect ticket prices.

The goal is to prepare the dataset for Machine Learning models and identify important patterns in flight pricing.

---

## Features in Dataset

- Airline
- Date of Journey
- Source
- Destination
- Route
- Duration
- Total Stops
- Additional Info
- Price (Target Variable)

---

## Tasks Performed

### Data Cleaning
- Removed unwanted columns
- Handled missing values
- Converted duration to numerical values

### Feature Engineering
- Extracted journey day and month
- Converted categorical features using encoding
- Converted duration into hours and minutes

### Data Analysis
- Checked correlation between features
- Identified important variables affecting price
- Visualized data using plots

---

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Goal
Build understanding of how airline, duration, stops, and route affect flight ticket prices.

This project is useful for building Machine Learning models for price prediction.
